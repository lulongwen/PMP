## [PMBOK](PMBOK/PMP.md)
* 五大过程组
  * 启动，规划，执行，监控，收尾

* 十大知识领域
  * 整合管理，范围管理，时间管理，成本管理
  * 质量管理，人力资源管理，沟通管理，风险管理
  * 采购管理，相关方管理

* 输入，工具，输出



## 1 启动过程组
* 设立目标，让项目团队有事可做


## 2 规划过程组
* 指定工作路线，让项目有法可依


## 3 执行过程组
* 按图索骥，让项目有法必依

## 4 监控过程组
* 车辆项目绩效

## 5 收尾过程组
* 相关方满意度，项目闭环，项目做得圆满
